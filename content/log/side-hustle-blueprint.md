## 用 Skill-Craft 方法论打造的技能

这个技能不是一次写成的。它用 [Skill-Craft](/projects#walkcraft-skill-craft) 方法论迭代了 8 个版本，6 个假设被证伪，最终稳定在 8 阶段流程。

## 8 阶段流程

```
阶段1: 摸底（双入口：供给出发 / 需求出发）
→ 阶段2: 身份匹配与商业可行性分析
→ 阶段3: 市场调研（双重验证）
→ 阶段4: 设计服务套餐
→ 阶段5: 客户筛选
→ 阶段6: 写平台文案
→ 阶段7: 商品展示素材
→ 阶段8: 输出文件
```

## 核心创新

### 双入口设计
用户可以从"我会什么"或"我看到什么市场机会"两个方向进入，在需求匹配点汇合。不管从哪条路来，最终都要走到"客户需求 ↔ 你的能力"匹配。

### 止痛药 vs 维生素
不是所有需求都一样。紧急痛点的服务最容易起步——客户主动搜索、马上付钱、不太比价。起步阶段优先选止痛药。

### 学习曲线定价
新手做一单比熟手慢是正常的，但客户拿到的是同样的结果，所以**不应该降价**。前期的时间亏损 = 投资给自己的学费。

### 信任四层体系
护城河解决"凭什么选你"，信任体系解决"凭什么信你"：外观信任 → 能力信任 → 保障信任 → 口碑信任。
