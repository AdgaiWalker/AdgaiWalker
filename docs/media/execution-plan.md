## 任务 1：iwalk.pro 首页加品牌卡片

**为什么做**：抖音视频发出去后，观众会点进 iwalk.pro。现在首页是个人博客风格，新访客不知道这个站在做什么。需要一张卡片，让人 3 秒内明白"这里有 AI 入门内容"。

**做什么**：在首页 Bento Box 中栏顶部插入一张品牌卡片，内容：
- 主标题：有点子，就试试
- 副标题：AI 入门到实战，按需求来
- 一个按钮：指向 /posts（文章列表）

**怎么做**：
1. 修改 `src/pages/index.astro`，在中栏顶部（现有内容之前）插入一张新卡片
2. 卡片样式跟其他 Bento 卡片统一：半透明背景、圆角、hover 阴影、drag-able
3. 按钮链接到文章列表页

**验收标准**：
- [ ] 打开 iwalk.pro 首页，中栏顶部能看到品牌卡片
- [ ] 卡片上能看到"有点子，就试试"和"AI 入门到实战"
- [ ] 点按钮能跳转到 /posts
- [ ] 卡片风格跟首页其他卡片统一，不突兀
- [ ] 移动端正常显示

## 任务 2：xyzidea.com 落地页

**为什么做**：抖音视频里会闪到 xyzidea.com 域名，观众可能会输入访问。如果 404 或空白，品牌形象受损。需要一个简洁好看的一页纸接住访问。

**做什么**：一个纯静态品牌落地页，内容：
- 品牌名：xyzidea
- 标语：有点子，就试试
- 一句话：AI 工具站，建设中
- 一个按钮：前往 iwalk.pro →

**怎么做**：
1. 在 NorthStar 仓库（`C:\Users\26296\Desktop\NorthStar`）中创建落地页
2. 纯 HTML + CSS，不需要框架，不需要后端
3. 部署到 Vercel，绑定 xyzidea.com 域名
4. 视觉风格：深色或浅色简洁，大字居中，一个按钮

**验收标准**：
- [ ] 浏览器访问 xyzidea.com 能正常打开
- [ ] 页面上能看到"有点子，就试试"
- [ ] 点按钮能跳转到 iwalk.pro
- [ ] 页面好看，不是白底黑字草稿
- [ ] 手机端正常显示

## 任务 8：录制抖音视频

**为什么做**：这是整个起手计划的引爆点。蹭抖音"晒作品"活动热度，展示项目 + 立品牌。

**做什么**：按规划文档第六节的视频脚本，录制一条 45-60 秒的竖屏视频。

**怎么做**：
1. 准备录屏工具（OBS / 系统录屏）
2. 浏览器调为手机竖屏尺寸（1080×1920）
3. 按脚本顺序录屏：
   - 快闪三个网站首页（0-3s）
   - iwalk.pro 拖拽 + 文章页（3-8s）
   - xyzidea.com 落地页（8-15s）
   - xyzidea.cn 落地页（15-20s）
   - GitHub 仓库列表截图快闪（20-25s）
4. 录制黑屏大字段落（25-30s）：可用剪辑软件加
5. 录制终端/AI 对话画面（30-38s）
6. 剪辑：拼接 + 加字幕 + 加背景音乐 + 加结尾卡
7. 发布：挂活动标签，写描述文案

**字幕文案**（按时间顺序）：
- "一个人，做了这些"
- "个人博客，用 AI 搭的"
- "AI 工具站，一个人从零写的"
- "共创社区"
- "还有框架、工具、方法论"
- "跨专业" / "不会写代码" / "九个月"
- "但我会想点子" / "AI 帮我做"
- "你也有点子？" / "来，我教你落地"
- "哟，你点子真多啊。—— xyzidea"

**发布文案**：
- 标题：一个人 + AI，九个月做了这些 #晒作品 #AI
- 描述：跨专业，不会写代码，但有点子。AI 帮我做。你也有点子？来 iwalk.pro，我教你落地。
- 标签：#晒作品 #AI #独立开发 #程序员 #跨专业

**验收标准**：
- [ ] 视频时长 45-60 秒
- [ ] 所有网站画面清晰可辨认
- [ ] 字幕跟画面同步
- [ ] 结尾有品牌卡："哟，你点子真多啊。—— xyzidea"
- [ ] 已挂活动标签
- [ ] 视频已公开发布
- [ ] 评论区已留一条引导评论
