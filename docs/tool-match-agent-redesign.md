### 任务

- [ ] 2.1 改造 `/api/match` 支持多轮消息

  - 验收：自然结束对话

- `src/pages/api/match.ts`

## 行为衔接逻辑

```
用户进入 /tools
  → 看到信号按钮："不确定用什么？说说你要做什么"
  → 点击 → 打开对话面板
  → Agent: "说说你要做什么，我来帮你找最合适的工具和资料。"
  → 快捷建议可点击快速开始
  → 用户描述需求
  → Agent: 追问缺失信息（场景/水平/目标）
  → 用户补充
  → Agent: 信息充足 → 输出匹配结果（工具 + 站内资料链接）
  → 用户可继续追问
  → 匿名需求自动进入选题库
  → Walker 在仪表盘看到需求趋势和选题建议
```

## 验收总标准

1. 全站零 Codex 特定文案
2. 匹配器可多轮对话
3. 推荐结果只引用站内资源
4. `/tools` 页面控制台零错误
5. 构建通过 + 部署成功
