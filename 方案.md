## P2 — 点赞功能

### 11. Supabase 点赞系统

**后端：**
- Supabase 创建 `likes` 表：`page_path`（主键）+ `count`（整数）
- 使用匿名 Key，无需用户登录
- 提供两个 API：获取点赞数 + 点赞（+1）
- 利用 Supabase RLS 策略限制每 IP 每页每天只能点赞一次

**前端：**
- 首页喜欢数替换为从 Supabase 实时读取
- 点赞按钮 + 动画效果
- localStorage 记录"已点赞"状态，避免重复点击
- 加载中/失败时优雅降级

**需要你做的事：**
- 注册 Supabase 账号，创建一个项目
- 把项目 URL 和 anon key 给我
